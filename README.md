# World Cup Pool Site

A simple static website for your prediction pool.

## Files
- `index.html` — main page
- `styles.css` — all styling
- `data.js` — editable pool data
- `app.js` — renders the page

## How to use
1. Open `data.js`
2. Replace the sample leaderboard, groups, bracket, and results with your own data
3. Upload the folder to GitHub Pages, Netlify, or any static hosting service

## Easy next step
If you want live scores from Google Sheets, replace `data.js` with a small fetch call to a published JSON/CSV export from your sheet.
