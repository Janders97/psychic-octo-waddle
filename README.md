# World Cup 2026 Pool

## Files
- `index.html`
- `styles.css`
- `app.js`

## Notes
- `WEB_APP_URL` in `app.js` should point to your Apps Script web app `/exec` URL.
- `PICKS_PUBLIC` controls whether the participant picks tab is shown as public or private.
- The site expects the live web app feed to return `leaderboardRows`, `knockoutRows`, `actualGroups`, and `picksRows`.
- The page refreshes every minute automatically.
